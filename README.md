# SFML-SineScroller
A sine scroller effect made in SFML with C++, adhering to Object Oriented Programming design principles. Made for monospace fonts but could be updated for non-monospace fonts.

https://github.com/user-attachments/assets/5f105aad-0776-44df-81a7-1c197785f92c

## Build
This program was built and tested with:  
* Windows 10 22H2
  * mingw-w64-ucrt-x86_64-gcc 15.2.0-1
  * mingw-w64-ucrt-x86_64-sfml 3.0.2-1

It will most likely work with newer or older versions and other operating systems, however, there may be some breaking changes that cause it not to build or run.
